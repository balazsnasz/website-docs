# Assets

WP Bones uses [gulp](https://gulpjs.com/) to compile and copy the assets from the `resources/assets` folder to the `public` folder. It supports:

- CSS (Less, Sass)
- Javascript (ES6, Typescript)
- ReactJS

## Installation

To compile the assets, you need to install the packages:

```sh npm2yarn copy
npm install
```

## Package.json

The `package.json` file contains the scripts to compile the assets:

```json
{
  "scripts": {
    "dev": "gulp watch",
    "build": "gulp build"
  },
}
```

To compile the assets, you need to install the dependencies:

```sh npm2yarn copy
npm run dev
```
All the assets in `resources/assets` are compiled and copied to the `public` folder.

The `gulpfile.js` in the root folder contains the tasks to compile the assets.
Feel free to modify the tasks to fit your needs.
